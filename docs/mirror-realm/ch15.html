第十五章〈1024年11月〉

台版 v3​

消息不是一個人說的。

是很多個人同時沉默。

早上回到公司，林昭明走過走廊。平時那種——有人倒水、有人講電話、有人在笑——沒有了。不是靜。是一種很熟悉的不對。

他坐下，開電腦。隔壁那個位置，有人在低聲跟另一個人說話。低到聽不清楚。但他聽到一個字。

電。

然後那個人停了。看了他一眼。聲音壓得更低。

林昭明沒有追問。他學會了不追問。

十一點左右，老王走過他的位置。放慢了兩步。沒有看他。低聲說了一句：「電相關那邊。不要問。」

林昭明看著他的背影走遠。

他開回email。照常工作。但腦子已經不在這裡。

電相關。三條Team。同一個老闆。

他知道那三條Team。他見過裡面的人。他知道他們做什麼、走哪套流程、交哪種報告。跟他自己的東西，很多重疊。

中午之前，消息在辦公室裡走了一圈。沒有人大聲說。沒有email。沒有公告。

有人死了。

林昭明坐在位置上。他沒有問任何人。因為每個人的表情都在說同一件事——「你知道就好，不要開口。」

下午兩點，老闆召集開會。不是關於那件事。但又是。

老闆站在白板前面。表情很正常。沒有沉重。沒有迴避。好像在說一件跟所有人有關但又跟所有人無關的事。

「工作歸工作。」

他頓了一頓。

「不應該給自己太多壓力。抗壓是你個人的問題。沒有人幫得了你。」

房間裡沒有人出聲。

林昭明看著老闆的臉。他在找一樣東西——任何一絲不自然。任何一刻的遲疑。

沒有。

他說話時很平穩。很真誠。好像他真心相信這句話。

一個人死了。他的回應是：你自己撐不住，是你的問題。

會議繼續。其他議程。林昭明坐在那裡，手放在桌面，沒有動。

下午三點。Team building。

沒有取消。

林昭明跟著去。因為不去更不對。

大家圍在一起。有人買了零食。有人搬了幾張椅子。氣氛是那種——刻意的正常。每個人都很努力「沒有發生過什麼」。

有人說起一部劇。《不夠善良的我們》。

「那部很好看。」

「人就是這樣嘛。」

「不夠善良所以才會這樣。」

有人接：「其實都是你吸引什麼就得到什麼。」

「對啊。你的能量不對，你過得不好，是你自己的問題。」

點頭。

林昭明坐在邊上。他沒有出聲。

話題轉了。有人說起帶孩子。

「我兒子鬧脾氣，我就不理他。」

「對，不要理他。等他自己冷靜。」

「冷處理。小孩要學東西嘛。你理他他就學不到。」

語氣是分享心得的語氣。很自然。很日常。好像在講食譜。

有人笑。有人點頭。輪流分享。

林昭明站起身。去廁所。

他在廁所站了一陣。看著鏡子。沒有想東西。或者想了很多但沒有一樣成形。

他回去。

他們還在說。

「有時候你要狠心一點。為他好。」

「我兒子試過哭了一個多小時，我都沒有理。最後他自己停了。」

「你看，有效吧。」

沒有人注意他走了。沒有人注意他回來。

他的存在和缺席，對這個對話的影響是零。

早上有人死了。

下午他們笑著說怎麼對自己的孩子用冷處理。

林昭明坐回去。他沒有話說。他不知道應該說什麼。他不知道為什麼覺得這兩件事有關係。理性上沒有關係。早上那件事是早上的事。下午的對話是另一件事。兩件事沒有連結。

但他坐在那裡，聽著他們說「不理他，他自己會學」，然後想起兩個小時前老闆說的那句話——

「抗壓是你個人的問題。沒有人幫得了你。」

同一個邏輯。

他不知道怎麼命名這樣東西。他只知道他坐在一群人中間，而他們的世界跟他的世界是不同的。不是他們壞。不是他們蠢。他們很真心。他們真的相信這些東西。

這個才是最恐怖的地方。

那晚他回到家。

太太在廚房。他換了鞋，坐在沙發上。沒有開電視。

太太走出來：「吃飯了嗎？」

「吃了。」

他沒有吃。

太太坐在他旁邊。他們沒有說話。過了一陣。

「你怎麼了？」

林昭明看著前面。

「聽到一個消息。」

「什麼消息？」

「很嚴重。」

太太看著他。等。

他沒有再說。

他不知道怎麼說。說「有人死了」？他不知道死者的名字。說「跟我做的東西有關」？他也不確定。說「我覺得這件事跟我的經歷有關」？他自己都還沒拼出來。

太太沒有再問。

他坐到很晚。太太進房睡了。他一個人在客廳。

然後他開了電腦。

PTT。

他搜尋。出來了。

有貼文直接點名。公司名。地點。部門。白紙黑字寫在那裡。

他一看就知道——因為那些名字他認得。那間公司。那個部門。那種壓力。

點名的貼文不多。但有。有人寫了幾段。寫得很直接。沒有閃爍。公司名全寫出來。哪條Team。什麼狀況。

林昭明看完，整個人定住了。

因為他認得那些字。不是認得那個字。是認得那個形狀。測試。壓力。二十四小時。沒有人理。

每一樣，他自己的身體記得。

他開始截圖。

他知道這些貼文的壽命。點名的——幾個小時。夠了。幾個小時已經夠。

有人問：「新聞呢？」

沒有人回答。

他截了幾張。存檔。繼續看。

他腦子裡閃過一樣東西。

不是1024年的事。是很久以前。

鵬廣。

★ 閃回B〈齋叔的桌子〉

鵬廣實業。林昭明入行第二年。

齋叔不是他取的綽號。整間公司都這樣叫他。因為他這個人很「齋」——不喝酒、不抽菸、不應酬。每天回來就工作。做完就走。幾十年。

齋叔是老闆。

不是打工的那種老闆。是白手起家的那種。鵬廣的電池設計，最早幾代，很多都是他的手筆。他在這裡做了幾十年。從一張桌子一個人開始。工廠從小到大，他全部見過。

林昭明那時還很年輕。他跟齋叔沒有深交。齋叔的位置在另一層。他們偶爾在pantry碰到。齋叔會說幾句——很老派的說法，說那些電池設計的事，說以前怎麼從頭做起，說材料怎麼挑。林昭明聽著。那種幾十年才有的穩，他聽得出來。

後來顧問來了。

林昭明那個level不知道詳情。他只知道：公司請了一批人進來。西裝。很多會議。有些會議他的老闆都進不去。

然後事情開始變。

第一樣他注意到的：維修。

維修本來就外包了。包工不包料。這個一直都是。但以前零件壞了，批個換件，正常流程。

現在不同。

有台機器壞了。零件要換。他去問。上面說：「不用換。你想辦法修回去。」

他看著那個零件。裂開的。物理性的裂開。不是調整一下就能解決的問題。

「這個不是調整的問題。要換。」

「你自己想辦法。」

想辦法。也就是——不承認要換，但機器還要能運作。你用膠帶也好，用鐵絲也好，你自己搞定。理論上所有東西還要正常運作。不是cut掉就算。是cut了資源但沒有cut標準。

同一個禮拜，隔壁那層樓有人在鋪新地板。

他看著工人搬磁磚進去。然後看回自己這邊。那台機器用膠帶貼著殼繼續用。零件不換。但地板要新。

然後設計部那邊出了問題。

林昭明不在設計部，但他聽到了。走廊的對話。Pantry的碎片。有人提過：某個設計有問題。不是小問題。但開完會之後，沒有人再提。

他不知道會議裡發生了什麼。他那個level沒辦法知道。

他只知道的是：齋叔變了。

以前齋叔開會，他說話很俐落。兩句講完重點，不用第三句。那種做了幾十年的人才有的力道。

現在齋叔開會少說話了。有時說到一半停了。看著桌面。然後說「沒事」。

有一次林昭明走過走廊，看到齋叔站在窗邊望著廠房。站了很久。不知道在想什麼。

林昭明走過去。沒有叫他。他覺得——這些是老闆之間的事。他太年輕。不關他的事。

認真做事的錢，全去做門面了。零件不換但機器要能運作。地板要新但設計有問題沒有人理。帳面要好看。

林昭明後來才知道：顧問進來的目的從來不是「改善」。是「把帳面做漂亮」。準備賣。

齋叔的心血——那些電池設計、那些從零開始的研發——在帳面上，只是一行折舊。

他信錯了人。不是因為蠢。是因為他以為大家都是來做事的。他一輩子做事，他以為其他人也是。

然後有一天早上。

林昭明回到公司。走廊的氣氛不對。有人在conference room裡，門關著。有人眼眶紅了。

他問隔壁同事。同事看著他。輕聲說：

「齋叔。今早。」

沒有人說「為什麼」。

沒有人需要說。

林昭明記得那天走過齋叔的位置。

桌面很整齊。太整齊。

一個杯子。一疊文件。一支筆。擺得很正。好像他整理過。好像他知道。

幾十年的東西，放在一張桌子上面。就這麼多。

三天後，公司請人來收拾。一個小時。文件入箱。桌面清空。

再過沒多久。整層樓圍了板子。裝修。

裝修完，他走過去看。齋叔那個位置沒有了。不是空了。是沒有了。整個layout改了。桌子多了。人多了。坐得密了。

齋叔做了幾十年的那層樓，現在坐著一批他不認識的人。沒有人知道這裡以前是什麼樣子。沒有人知道有張桌子放過幾十年的東西。

連個位置都沒有留。

林昭明那時可以走開。

他的確走開了。

他不是齋叔的人。他不知道具體的會議。他不知道顧問跟齋叔之間說過什麼。他只知道一個一直站得很穩的人，慢慢沉下去，然後有一天早上沒有了。

他去買了杯咖啡。喝完。回去工作。

一個禮拜之後，沒有人再提。

公司繼續運作。顧問繼續開會。地板繼續鋪。零件繼續不換。

林昭明沒有想太多。他那時覺得——這是齋叔的處境。不是我的。他是老闆。老闆的事，我搞不定。

他就這樣跟自己說。

他相信了很多年。

閃回結束。

1024年11月。林昭明看著電腦螢幕。

PTT上面的字。同樣那些字。

「壓力」。「沒有人理」。「測試」。「二十四小時」。

他認得。不是因為他見過這些字。是因為他走過那條路。

齋叔的桌子。三天清空。整層樓裝修。連位置都沒有留。沒有人再提。

這次也是。

但這次不同。

齋叔那次，他走開了。他可以走開。因為他不是裡面的人。他還年輕。他不懂。他有藉口。

這次——死的那個人在同一個老闆底下。同一套手法。同一種壓力。PTT上面描述的每一個細節，他自己的身體記得。

他走不開。

不是因為勇敢。是因為他已經在裡面。

那個工程師死了之後，我才慢慢明白。

我一直以為 null pointer 問題只是被壓住。有人不想理。有人覺得不重要。有人想拖。我以為最差的情況是「沒人理」。

我沒想過。

死人之後，這個問題不是「被壓住」。是「必須不存在」。

如果 null pointer 問題被確認存在——那死人事件就可能有關聯。有關聯就有法律責任。有法律責任就公司上上下下都有事。

所以這個問題，必須從來沒存在過。

不是「不敢提」。是「從來沒這件事」。

我記得那幾天。有些 ticket 不見了。有些 comment 被改過。有人問「那個 null pointer issue 現在怎樣？」，對方看著他，好像聽不懂他在說什麼。

我以前以為「壓住」已經是最差。

不是。

最差是「從來不存在」。

因為壓住，還代表有一樣東西在那裡。

從來不存在，代表連那樣東西都沒了。連你記得這件事本身，都變成你一個人的問題。

那幾天，林昭明每晚回到家就開電腦。

PTT。DCard。論壇。

第一晚——點名的貼文已經沒有了。幾個小時。他那晚截到的東西，現在找回同一條link，「此文章已被刪除」。但不點名的還在。說壓力的、說測試的、說「沒有人理」的。沒有寫公司名。所以留得久一點。

第二晚，少了。有些板主出來說「這個不符合板規」。

第三晚。換了關鍵字才找到。殘留的貼文。有人引用過的截圖。但原文已經沒有了。

第四晚。找到一個。發文日期對。裡面有人問「新聞呢？」下面沒有人回答。貼文停在那裡。沒有人再說。

不到一個禮拜。全面封鎖。

點名的——幾個小時就沒有了。不點名的——幾天。但最後都沒有了。

好像那個人從來沒有存在過。

林昭明坐在螢幕前面。他找到的不只這一件。

同一類設備。兩個月內。不只兩件。

他看到第三件的時候，他選擇了不繼續看。

不是找不到。是他選擇停下來。

每多找一個，他就多知道一件他承受不住的事。每一個貼文背後都是一個人。一個有名有姓、有同事認識、有人問「新聞呢」然後沒有人回答的人。

他坐在那裡。螢幕的光映在他臉上。家裡很靜。太太睡了。

他合上電腦。

不是不在乎。

是「我已經知道夠多了。再多，我承受不住。」

螢幕暗了。家裡很靜。窗外有車聲，很遠，然後消失了。

他坐在那裡。沒有開燈。沒有動。

最深的消失，不是被遺忘。

是被記得，但記得的方式，令他的死永遠跟系統無關。
